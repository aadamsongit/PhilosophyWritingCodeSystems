## Frontend Fundamentals

Frontend development consists of many layers. JavaScript is the industry standard programming language for the web. React is the industry standard library for building modern user interfaces. Node.js is the industry standard JavaScript runtime.

When people talk about "fundamentals," they are often referring to JavaScript itself: its runtime behavior, asynchronous execution, objects, functions, closures, scope, and related language concepts. TypeScript builds on those fundamentals by making contracts and intent explicit.

## JavaScript Micro-Builds Are Great for Fundamentals Practice

Small JavaScript projects are an excellent way to reinforce the fundamentals. Build simple applications, experiment with the DOM, work with APIs, practice asynchronous programming, and reason about state without the additional abstraction of a large framework.

The same applies to system design and data structures and algorithms. Python is often a pleasant language for learning DSA because of its concise syntax, but solving problems in JavaScript or TypeScript reinforces the runtime behavior and state transitions you'll encounter in frontend development. Practicing in both languages has value for different reasons.
